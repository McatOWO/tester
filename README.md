# EFT タスク管理アプリ

Escape From Tarkov のクエストを、トレーダー・マップ・進捗ステータスで管理するシンプルなWebアプリです。

## 主な機能

- タスク追加（クエスト名 / トレーダー / マップ / 目的 / 優先度 / ステータス）
- ステータス更新・削除・完了タスク一括削除
- トレーダー・マップ・ステータス・キーワード検索による絞り込み
- 優先度とステータスを考慮した並び替え（進行中 & 高優先度が上位）
- 全体進捗バーと完了率表示
- 初期データとして主要タスク群を大量投入（Prapor / Therapist / Skier / Peacekeeper / Mechanic / Ragman / Jaeger / Fence）
- ローカルストレージ保存（ブラウザ再読み込み後も保持）

## 使い方

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いて使用してください。
